Version 1.7
— October 3, 2016

This version change brings new features in response to some of the most popular requests received from users in the last few months:

 

Customizer with Live Preview over the old Theme Options panel

Customizing the theme is now a much better experience using the native WordPress Customizer. Beside having simplified, easy-to-follow settings, you can now live preview your changes before saving the output.

If you’re not familiar with it yet, open it up by going to Appearance > Customize in your administration dashboard. This improvement is meant to save a few extra clicks and page refreshing time, which will result into better productivity, but don’t worry if you’re the nostalgic type, the old Theme Options panel is still accessible from the same location: Growrich > Theme Options.

 

Improved Font Options

You can now easily set different global font sizes for both the Headings and the Body Text. Find the new option under Appearance > Customize > Fonts.

 

New Premium webfonts, free with Growrich

Two new premium webfonts, Neue Einstellung and Radnika designed by Hanken Design Co were included and they now bring more awesomeness to Growrich.

 

A new plugin/add-on to integrate Adobe Typekit webfonts

Bring even more font awesomeness to Growrich by integrating it with Adobe Typekit. 

 

Favicon Source

Part of making the switch to the native Customizer was to remove the old option for uploading a favicon file and leave the theme to rely on the native option of WordPress, which can be found under Appearance > Customize > Site Identity.

Version 1.6.9 
— 21 July 2016

Performance Improvements / Faster Loading Time

After valuable user feedback, extensive research and testing, we spotted several opportunities to make Growrich faster and more efficient. The goal was to keep the theme and its code as clean and fast as possible while packing the same awesome features people love without slowing things down. By minifying JavaScript and CSS code, we managed to improve performance and speed up page loading, making both visitors and search engines happy.

  • 12 JS files merged and minified into a single file;
  • 16 CSS files merged and minified into a single files;

Less files means less server requests, which means faster loading times.

There's a New Plugin in Town

After updating the theme to 1.6.9, a notification will show up on your administration dashboard screen inviting you to install a new plugin called: Get Bowtied Tools. This is a small plugin that handles the automatic theme updates, your products key and the demo content installer. Part of making the theme lighter and faster was to move this back-end functionality outside the theme, into a plugin. So, just go ahead and install the plugin as well after updating the theme and stay worry free, nothing changes as far as you should be concerned as a site administrator.
