Portfolio — Adding New Items / Projects

Adrian
Updated December 08, 2016 14:58

Navigate to Portfolio → Add New to start creating projects.

The portfolio projects are being just like the rest of the pages, you can combine the elements available using the Visual Composer's drag-and-drop interface. This means each project can be totally custom.
